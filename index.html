Everything you need to sing along

🎵

Automatic playback detection

Built on Windows SMTC. Works with Spotify, Apple Music, YouTube Music and any player that supports media keys.

🔎

Multi-source lyrics search

Searches LRCLIB and NetEase, then auto-picks the best match using quality ranking.

🌐

Bilingual display

Shows the original line with its translation stacked below, so you can follow both at once.

✨

DeepL translation fallback

When a source has no translation, DeepL machine-translates on the fly — bring your own free API key.

🖥️

Desktop overlay

Transparent, click-through, always on top. Karaoke-style progress fill and drag to reposition.

💾

Offline cache

Found lyrics and translations are stored in SQLite for instant, offline display on replay.

⚙️

Tray controls

Toggle the overlay, nudge sync offset by ±0.5s, search manually, and launch on startup.

🔄

Automatic updates

Delta updates via Velopack keep you on the latest version automatically.

See it in action

Lyrics fade in over your desktop and fill line by line as the song plays.
